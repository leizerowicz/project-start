## Usage Instructions for AI Agents

### Before Making Decisions
1. **Read Current State**: Review project phase, priorities, and constraints
2. **Check Constitutional Compliance**: Ensure decisions align with constitutional principles
3. **Consider Context**: Reference successful patterns and lessons learned
4. **Validate Approach**: Confirm approach matches current architectural constraints

### When Updating Context
1. **Update Relevant Sections**: Modify applicable sections with new information
2. **Maintain Consistency**: Ensure updates don't conflict with existing context
3. **Document Rationale**: Include reasoning for context changes
4. **Sync Systems**: Trigger context synchronization after updates

### For Complex Decisions
1. **Consult Constitutional Memory**: Review governance principles and precedents
2. **Check Lesson Memory**: Learn from previous similar situations
3. **Consider Multi-Agent Impact**: Assess impact on other agents and systems
4. **Document Decision**: Update project memory with decision and rationale

This memory system eliminates the need to repeatedly explain project context to AI agents while ensuring constitutional compliance and continuous learning.
